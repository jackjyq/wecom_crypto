# 企业微信加解密库

重置企业微信[官方推荐的加解密库](https://github.com/sbzhu/weworkapi_python)，以支持最新的 Python 特性。

## 依赖安装

![](https://img.shields.io/badge/python-3.10-yellow)

```shell
pip install pycryptodome
```
